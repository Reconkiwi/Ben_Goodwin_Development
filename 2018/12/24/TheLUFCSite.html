The LUFC News project was a rebrand of Alex Matthews' fan made community social media group and website. I had a direct role in re-developing Alex's website by creating a new Wordpress theme for him from scratch which included a login system and live chat room.

The LUFC News Site - Y3S1

The Brief

During this semester we were required to organise or create our own briefs to complete over a 12 week period. As my development projects for my work placement had not begun yet, I decided to source an alternative project to work on for the rest of the semester rather than waiting for my work placement project to get up and running.
Alex Matthews, a client that I had worked with in my previous year of study (developing a prototype for his fan based Leeds Football news app), was looking to rebrand his community and identity starting with his social media platforms and eventually moving up to his website lufcnews.co.uk.
I was contacted after the rebrand of the company in the hope that I would build the site based on Alex’s LUFC News WordPress theme. Here is how the site looked before development started:

Alex had said to me that the website was no longer gaining consistent traffic and that the design had become outdated as he did not have the time or necessarily the skills to keep updating the templates.
It was clear that Alex’s website had become a little dated, so it was important for me to gain an understanding of precisely what he required from the sites redevelopment.

Initial Problems and Solutions

There were two main initial problems before development. Firstly, I had never developed using WordPress before, therefore, I had a fairly steep learning curve to take into account so that I could manage my time accordingly and I would need to read a few tutorials or speak to a WordPress specialist on my course as well to ensure I fully understood the WordPress theme development. Secondly, having looked through the code of Alex’s original theme I felt that it wouldn’t be possible to rebuild the site based off of his theme. This meant that I needed to discuss building a new theme from scratch with Alex as well as needing to find the quickest possible solution to building a WordPress theme from scratch. For this I needed to come up with a series of ideas:

Idea 1 - Building Pages and Directories from Scratch

Initially I had considered creating the pages, configurations and directories from scratch. This would have meant that I would have a better understanding of WordPress’ systems and setup. As well as this it would mean that all of the code on the theme would be mine which would help with development further down the line as everything would be setup in the same format.

Problems With Idea 1

The main issue that i faced with this idea was the time that it would take me to get a blank WordPress theme up and running. Alex was hoping to see progress within the first two weeks of development which meant that I didn’t have a large window to get some of the pages completed.

Idea 2 - A Course Mates Blank Template

The second idea was to build the site using a course mates blank WordPress theme. The benefits to this being that the blank template was being used commercially already, therefore, I was aware of the templates potential as I could see how it had been implemented into many sites.

Problems With Idea 2

The downside to this idea was that the blank template needed to be reformatted to ensure that it was not breaching any of the themes licensing agreements. This would have taken quite a lot of time and would have potentially caused more problems rather than solve any.

Idea 3 - Bones Theme

The third idea was to look into using the Bones WordPress theme. Bones is a HTML5, mobile first, blank theme used for rapid development. Bones also came with a free license meaning that I could use it for Alex’s site free of charge

Problems With Idea 3

My biggest problem with Bones was that I couldn’t get on with the documentation. This was a really large issue for me considering my lack of knowledge in Wordpress development.

Idea 4 - Underscores theme

Having looked into various solutions to my problem, and still being unsatisfied with my options, a masters student suggested Underscores to me. Underscores is a little like Bones, however, it aims to use minimal CSS which allows for developers to tear the base theme apart and completely rebuild it from the ground up. As well as this Underscores comes with a ready to go translation package and custom 404 pages to help get developers started.

The Final Solution

I ended up opting for Underscores rather than any of the other options, this was mainly due to the large amounts of documentation and tutorials based around the theme. I felt that this would be really beneficial to me as I could use these to increase my development efficiency.

Development Tools

To organise myself efficiently, I also made a list of the development tools that I would need to use:

MAMP

MAMP is a free local environment server that is often used when working with languages such as PHP. As WordPress sites are built using PHP, MAMP was the perfect tool for the development of Alex’s theme.

MySQL

WordPress also requires databases to store the posts, users and comments. Therefore, MySQL, an open source relational database system was a good option for the development of this. As well as this MAMP has phpMyAdmin, an administration tool for MySQL, integrated within it, therefore, it would be efficient to use this tool.

HTML, CSS AND JS

Like most other websites, HTML, CSS and JS are imperative to web development. Therefore, I would be using these for development as well.

PHP

PHP is a server-side coding language for web development and is directly used during WordPress theme development. It was clear that I would need to use this during my development.

Initial Development

Before I begin discussing the initial development of the the LUFC News theme, I will outline the WordPress plugins that I have used to help with my development and an explanation as to why I have used them.

WordPress Plugins

  • Advanced Custom fields
Likely to be the most important plugin that I used during my development. Advanced Custom Fields are used to help improve the efficiency of creating user input fields for WordPress themes. This plugin was recommended to me by a developer on my course as he had said to me that the studio he had been working in had been using this plugin for all of their commercial products.
  • Contact Form 7
A simple but effective form plugin, however, it comes setup with an anti-spam software to help protect the client from spam mail.
  • Gmail SMTP
The WordPress emailing system fails when running on local servers, therefore, Gmail SMTP was used to help test the contact forms and user logins of the site.
  • iFlyChat
A simple chat plugin with chatrooms and embed system for multiple platforms. As well as this iFlyChat comes with its own dashboard where the administrator of the site can monitor what is being discussed in the chat with the ability to set banned words as well as kick, ban or ban the IP of any user they wish.
  • Instagram feed
A simple instagram plugin to show the clients feed.
  • Menu Image
A plugin to add images or logos to the WordPress menus.
  • Soccer Info
A plugin to display the league tables of specific football tournaments.
  • WP Rest API
JSON based rest API for Wordpress, used for the LUFC App (see LUFC App blog post for more details).
  • Ultimate Responsive Image Slider
Simple image Slider used to display image galleries for all platforms.
  • JWT Authentication for WP-API
Extends the WP Rest API to allow for JSON Authentication (also used for the LUFC App).
